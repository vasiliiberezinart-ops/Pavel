# СВОБОДА — клип Павла Фахрутдинова

Препродакшн-сайт клипа. Концепт — лагерь для беженцев, тихая клоунада. Семь раскадровок по структуре трека.

## Стек
Чистая статика — `index.html` + `styles.css` + `script.js`. Без билда, без зависимостей. GitHub Pages.

## Структура
```
.
├── index.html        # одна страница, RU/EN toggle
├── styles.css        # тихая клоунада: бумага, мелки, наивная типографика
├── script.js         # i18n-словарь + click-to-seek по таймингам
├── assets/
│   ├── svoboda.mp3   # трек
│   └── waveform.png  # сгенерированный волновик
├── drawings/         # 01.png … 07.png — раскадровки (заливаются после Qwen)
└── PROMPTS.md        # готовые промпты для каждого кадра
```
